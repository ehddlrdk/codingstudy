### 문자열 추가
```c
string s1("Welcome");
s1.append(" to C++") // s1에 " to C++" 문자열 추가 --> Welcome to C++ 출력

string s2("Welcome");
s2.append(4, 'G'); // s2에 'G' 문자를 4개 추가 --> WelcomeGGGG 출력
- 문자를 뒤에 붙여야할 때, 나는 보통 s2.push_back('G')와 같이 사용했음
```

### 특정 문자열 제거 - erase
```c

string s2("12345678");

//매개변수가 한 개인데, 매개변수가 숫자일 때

string s2("abcdefgh");
s2.erase(2); //"ab"
```
